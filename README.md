# ATM app

### Installing and running

Install all dependencies:
```
$ npm ci
```

Then run:
```
$ npm run start:api
```

Followed by:

```
$ npm start
```

Open:
```
http://localhost:3000
```

## Testing

To run all tests run:
```
$ npm test
```

## Build

To build the app run:

```
$ npm run build
```
